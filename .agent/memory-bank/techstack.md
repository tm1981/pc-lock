# Tech Stack & Dependencies

## Runtime Requirements

- **Python 3.10+** (uses `|` union types, dataclasses)
- **Windows only** (Win32 APIs)

## Dependencies

```
screeninfo==0.8.1     # Multi-monitor detection
pystray==0.19.5       # System tray icon
pywin32==306          # Win32 API (registry)
pycaw==20240210       # Audio muting via COM
comtypes==1.2.0       # COM interface for pycaw
pillow==10.4.0        # Tray icon rendering
customtkinter==5.2.2  # Modern dark theme UI widgets
winotify==1.1.0       # Windows toast notifications
```

## Standard Library Usage

| Module | Usage |
|--------|-------|
| `tkinter` | GUI windows, dialogs |
| `ctypes` | Win32 API bindings |
| `threading` | Background scheduler, watcher |
| `subprocess` | Lockscreen process spawning |
| `http.server` | REST API server |
| `json` | Config, schedule, API payloads |
| `hashlib` | PBKDF2 password hashing |
| `secrets` | Random salt generation |
| `argparse` | CLI argument parsing |
| `winreg` | Startup registry entries |

## Config Files

| File | Location | Format |
|------|----------|--------|
| `config.json` | `%LOCALAPPDATA%/PC-Lock/` | JSON (plaintext) |
| `schedule.dat` | `%LOCALAPPDATA%/PC-Lock/` | Binary (DPAPI encrypted) |

## Build Tool

**PyInstaller** for single-exe packaging:

```powershell
pip install pyinstaller
python make_icon.py              # Generate assets/pclock.ico
pyinstaller --noconfirm --onefile --windowed \
  --name pclock \
  --icon assets\pclock.ico \
  --add-data assets\pclock.ico;assets \
  main.py
```

**Output**: `dist/pclock.exe`

## Assets

| File | Purpose |
|------|---------|
| `assets/pclock.ico` | Window/tray icon (generated by make_icon.py) |
